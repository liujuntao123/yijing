# 东方古典人文与侘寂风格网页设计指南 (Design System Guide)

本设计指南旨在定义一种将**东方禅意美学（Wabi-Sabi）**、**经典纸质印刷排版（Editorial Print）**与**现代网页交互技术**相融合的视觉体系。通过克制的色彩、考究的留白和严谨的字学排版，为用户提供一种静谧、深邃且具有呼吸感的阅读体验。

---

## 1. 核心设计哲学 (Core Philosophy)

*   **克制与留白 (Restraint & Negative Space)**
    *   “无即是有”。不以多余的装饰填满屏幕，而是通过慷慨的物理空白（Margin & Padding）引导视线，给阅读者留出思考与呼吸的空间。
*   **接受岁月的质感 (Appreciating Patina & Imperfection)**
    *   摒弃绝对的高光、完美的圆角与生硬的几何饱和度。选用带有微弱纸张肌理的底色、柔和的边界线和不完全对称的布局。
*   **纸质阅读的回归 (The Return to Print)**
    *   模拟传统木版水印、线装书和现代高档艺术纸书籍的排版特征。重视字距、行高、首字下沉（Drop Caps）与边栏批注。
*   **人文主义的温润 (Humanist Warmth)**
    *   冰冷的机器灰色被具有泥土、沙石温度的暖灰与炭黑取代，偶有如落款印章般的朱砂红作为视觉焦点，传递温存的东方情感。

---

## 2. 色彩系统 (Color Palette)

色彩不应夺目，而应作为内容的画布。所有颜色均提取自自然界、陶器、宣纸以及古典水墨。

```
+-------------------------------------------------------------+
|  素纸 (Paper)      #FAF8F5  | 核心背景色，模拟高档未漂白艺术纸或手工宣纸的色泽    |
+-------------------------------------------------------------+
|  粗陶 (Clay)       #F3EDE2  | 辅助色/卡片背景，模拟自然泥土或麦麸的温暖温润感     |
+-------------------------------------------------------------+
|  暖炭 (Charcoal)   #1F1B18  | 主文本色，避免使用纯黑，带有温度的微暖深炭灰       |
+-------------------------------------------------------------+
|  细沙 (Sand)       #6E665A  | 次要文本/辅助框线，石质风化后的沙灰色              |
+-------------------------------------------------------------+
|  朱砂 (Seal Red)   #962E2A  | 点缀色，如印章朱泥，仅用于关键状态、指引或强强调项   |
+-------------------------------------------------------------+
```

### 搭配比例建议
*   **背景色 (Background):** `素纸 (Paper)` 占 75%
*   **辅助块面 (Component/Card):** `粗陶 (Clay)` 占 15%
*   **主文本 (Text Primary):** `暖炭 (Charcoal)` 占 8%
*   **辅助文本与框线 (Text Muted/Borders):** `细沙 (Sand)` 占 1.8%
*   **视觉焦点 (Accent):** `朱砂 (Seal Red)` 占 0.2%（极其克制）

## 3. 字体与排版系统 (Typography)

字型是传递人文情感的最直接媒介。本系统推荐使用高品质的**宋体/明体**作为中文主干，辅以经典的**西文衬线体**表现数字与西文字符。

### 3.1 字体选用 (Font Stack)
*   **中文主设字体 (Primary Chinese):** 
    `"Noto Serif SC" (思源宋体)`, `STSong (华文宋体)`, `SimSun (宋体)`, `serif`
    *   *说明：* 避免在古典内容中使用无衬线字体（如微软雅黑、苹方），宋体特有的横细竖粗更具木刻雕版印刷的韵味。
*   **西文与数字字体 (Western & Numbers):** 
    `"Playfair Display"`, `"Cinzel"`, `Georgia`, `serif`
    *   *说明：* 西文衬线体与中文宋体在气质上高度契合，尤其在章节编号（如 01, 02）时能展现出极佳的典雅感。

### 3.2 字学排版规范 (Typography Specs)
*   **行高 (Line Height):** 
    *   正文阅读建议保持在 `1.75` 到 `2.0` 之间（即 Tailwind 中的 `leading-relaxed` 或 `leading-loose`）。
*   **字距 (Letter Spacing):** 
    *   大标题与段落标题建议增加字距（如 `tracking-widest`），赋予文字庄重、舒缓的仪式感。
*   **最大段落宽度 (Max Width):** 
    *   为了保证视线折返的舒适度，正文容器宽度通常建议限制在 `max-w-2xl` 到 `max-w-3xl`（约 650px - 750px 之间）。

## 4. 网格、空间与质感 (Grid, Spacing & Texture)

### 4.1 框线控制 (Borders)
*   避免使用实线包围每一个元素。推荐使用极细（1px）且低饱和度的边框或虚线（如 `border-sand/10` 或 `border-dashed`）。
*   多用“单边线”（如仅在左侧或底部加线）来模拟线装书的格线，而非封闭式方框。

### 4.2 质感叠加 (Tactility)
*   **微噪点底纹：** 可以在网页顶层或背景层叠加一层极低透明度（1% - 2%）的 SVG 噪点，以消除数字屏幕纯色块的冰冷感，模拟纸张的纤维质感。
*   **自然投影 (Shadows)：** 
    *   尽量避免使用现代扁平化大面积发散的彩色投影。
    *   若必须使用阴影，建议使用极硬、偏小的暗色投影（类似墨迹干涸或石雕压痕），或者完全不使用投影，依靠色块冷暖交界来区分层次。

---

## 5. 关键组件设计标准 (Component Specifications)

### 5.1 标题区 (Headers)
*   **结构：** 序号 + 垂直分隔（或印章）+ 主标题 + 英文副标题。
*   **排版示例：**
    ```html
    <div class="space-y-2">
        <span class="text-xs tracking-widest text-seal font-bold">第一章 · 卦辞</span>
        <h2 class="text-3xl font-bold tracking-widest text-charcoal">乾为天</h2>
        <p class="text-[10px] tracking-widest text-sand uppercase font-display">The Creative Qi</p>
    </div>
    ```

### 5.2 释义卡片 (Interpretation Cards)
*   使用 `bg-clay/30`（极淡的燕麦色）作为卡片背景。
*   采用左侧单边朱砂红实线（`border-l-2 border-seal`）引导视觉，模拟古书旁批。

### 5.3 符号与插图 (Iconography & Diagrams)
*   **传统符号（如八卦、太极、爻线）：**
    *   一律采用 SVG 绘制，确保线条宽度与周围字体笔画（如宋体的横线）粗细相近。
    *   爻线（阴爻、阳爻）的高度不宜过大，间距应保持呼吸感，使其看起来像雕版印刷的印迹而非数字化的条形码。

---

## 6. 微交互与动效规范 (Transitions & Motion)

侘寂美学下的动效应当是**隐秘、缓慢、自然**的，绝不能有突兀的弹跳或极速的闪烁。

*   **淡入淡出 (Fade-in)：**
    *   页面加载或状态切换时，推荐使用时长在 `500ms - 800ms` 的渐显效果，并配合微弱的位移（如向上位移 4px），模拟宣纸吸墨浸润的过程。
*   **悬停反馈 (Hover Effects)：**
    *   按钮或链接在 Hover 时，不建议做大面积的背景色突变，而推荐采用文字颜色平滑过渡（如 `text-sand` 渐变为 `text-seal`），或下划线自中间向两端缓缓展开。

## 7. 前端开发工程实践 (Tailwind CSS Integration)

为了方便快速在项目中还原此设计，建议将以下配置写入您的 `tailwind.config.js` 文件中：

```javascript
module.exports = {
  theme: {
    extend: {
      fontFamily: {
        // 宋体/明体作为核心中文字体，Playfair/Cinzel作为经典数字和英文
        serif: ['"Noto Serif SC"', 'STSong', 'SimSun', 'serif'],
        display: ['"Playfair Display"', 'Georgia', 'serif'],
        classic: ['"Cinzel"', 'serif'],
      },
      colors: {
        paper: '#faf8f5',     // 宣纸暖白（底色）
        clay: '#f3ede2',      // 粗陶暖沙（卡片与容器）
        charcoal: '#1f1b18',  // 暖炭黑（主文字，柔和不刺眼）
        sand: '#6e665a',      // 细沙灰（辅助文字、细边框）
        seal: '#962e2a',      // 印泥朱砂（状态、焦点、印章）
      },
      letterSpacing: {
        widest: '0.2em',      // 更加开阔的字间距，适合古典标题排版
      },
      lineHeight: {
        relaxed: '1.8',       // 舒缓的正文行高
        loose: '2.2',
      }
    },
  },
  plugins: [],
}
```

### 设计落地检查清单 (Design Checklist)
1. **背景检查：** 是否使用了纯白色 (`#FFFFFF`)？（若是，建议替换为 `#FAF8F5`）。
2. **文字检查：** 是否使用了纯黑色 (`#000000`)？（若是，建议替换为 `#1F1B18`）。
3. **字体检查：** 标题与正文是否正确应用了思源宋体等衬线字型？
4. **留白检查：** 每一个核心区块之间，是否留出了足够的物理空白（建议使用 `py-16` 或 `py-24`）？
5. **线条检查：** 页面中是否存在过于显眼、高对比度的粗实线？（若是，建议将其淡化为 `border-sand/10` 级别的细线）。
