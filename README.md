# 周易 64 卦查询

这是一个用来查询和阅读《周易》六十四卦的小工具。

我做这个项目，主要是因为平时想查某一卦、某一爻，或者想看一个爻变之后会变成哪一卦时，资料往往比较分散。传统文本、白话解释、象传、彖传、爻辞、变卦关系都有，但需要在不同页面之间来回找，不太方便。

所以我想把这些内容整理成一个可以直接使用的页面：既保留《周易》原文和常见解读，也让卦象、爻位、八卦和变卦之间的关系更直观。它不是为了把《易经》讲得很玄，而是希望在查阅和理解时少一点阻力。

## 主要功能

- 查看六十四卦的卦名、卦象和简义。
- 通过八卦入口快速切换到对应的重卦。
- 搜索卦名、卦象或简义，快速找到想看的卦。
- 阅读每一卦的卦辞、彖辞、象辞、白话解释和相关解读。
- 点击六爻中的任意一爻，查看对应爻辞。
- 查看当前爻变化后得到的变卦，并可以直接跳转到变卦。
- 将原始 Markdown 内容整理成结构化数据，方便页面展示和后续扩展。

## 本地运行

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```

运行测试：

```bash
npm test
npm run test:e2e
```

## 数据来源

项目中的六十四卦文本整理在 `yijing64/` 目录下，页面使用的结构化数据位于 `src/data/yijing.json`。数据整理脚本放在 `scripts/` 目录中。
